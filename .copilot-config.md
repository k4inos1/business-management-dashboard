# GitHub Copilot Automation Configuration

## 🤖 Copilot Automation Rules for business-management-dashboard

### PR Review Automation

**Triggers:**
- `review: auto` → Genera automáticamente sugerencias
- `ui: check` → Verifica UI/UX consistency
- `types: check` → Verifica TypeScript types
- `lint: check` → Ejecuta ESLint

### Auto-Merge Rules

```yaml
Auto-merge cuando:
  - ESLint pass ✅
  - TypeScript check pass ✅
  - Build success ✅
  - No breaking changes ✅
```

### Copilot Commands

```
@copilot review: ui                # Review UI changes
@copilot check: types              # TypeScript check
@copilot suggest: ux-improvements  # UX suggestions
@copilot generate: storybook       # Generate stories
```

### Auto-Labels

```yaml
UI changes → 'ui'
Business logic → 'business-logic'
Performance → 'performance'
Accessibility → 'a11y'
Testing → 'testing'
Documentation → 'documentation'
```
